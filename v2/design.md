# Epic v2 language design

v2 currently starts from the v1 language and has not introduced its own new language features yet.

The v2 source tree is reserved for the first compiler sources that are compiled by the v1 compiler. Until v2 grows distinct semantics, treat `../v1/design.md` as the inherited language design and keep this file focused on differences introduced by v2.
